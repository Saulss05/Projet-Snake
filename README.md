Ce projet a été réalisé dans le cadre académique du cours de programmation impérative en C. Le but est d'implémenter le jeu Snake bien connu. 
Pour cela, nous avons utilisé des concepts en C comme les listes chainées, nous avons du gérer l'allocation mémoire, essayé de soigner l'interface,
et même ajouter quelques variantes au jeu originel.
